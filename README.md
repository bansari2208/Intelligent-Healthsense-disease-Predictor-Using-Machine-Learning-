# Disease Predictor using Machine Learning

This project predicts diseases using machine learning models based on symptoms input.

How to run the project ?

Step 1 : Open Path of your file in Terminal
exg : cd C:\Users\SVI\Desktop\Disease-Prediction-using-ML

Step 2 : Activate virtual environment 
venv\Scripts\activate

Step 3 : Run server 
python manage.py runserver
